# 🦁 FinVi — Finanças Inteligentes

App de gestão financeira pessoal com assistente de IA.

## ✨ Funcionalidades

- **Dashboard** com saldo, receitas e despesas
- **Extrato** com busca e filtros por categoria
- **Orçamentos** com gráfico e alertas
- **Metas financeiras** personalizáveis
- **Cartões** de crédito com limite e fatura
- **Adicionar transações** (despesas e receitas)
- **Finn IA** — assistente financeiro com IA real

---

## 🚀 Como publicar no Vercel (passo a passo)

### 1. Instalar dependências localmente
```bash
npm install
npm run dev   # testar antes de publicar
```

### 2. Criar conta no GitHub
- Acesse [github.com](https://github.com) e crie uma conta gratuita
- Crie um repositório novo chamado `finvi`
- Faça upload de todos os arquivos deste projeto

### 3. Criar conta no Vercel
- Acesse [vercel.com](https://vercel.com)
- Clique em **"Sign Up"** e entre com sua conta GitHub
- Clique em **"Add New Project"**
- Selecione o repositório `finvi`
- Clique em **"Deploy"** — pronto! 🎉

### 4. Seu app estará disponível em:
```
https://finvi.vercel.app  (ou similar)
```

## 📱 Instalar como app no celular (PWA)

**Android (Chrome):**
1. Abra o link do site no Chrome
2. Toque no menu (⋮) → "Adicionar à tela inicial"

**iPhone (Safari):**
1. Abra o link no Safari
2. Toque em Compartilhar (□↑) → "Adicionar à Tela de Início"
